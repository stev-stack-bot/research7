# Manage Referrals

Using an `auth` code, you can set which referral code to use on an account. To do so, use [referral\_use](https://apidocs.lighter.xyz/reference/use_referral_code) indicating your L1 address and the referral code you want to assign to it.

Once done, you can check [userReferrals](https://apidocs.lighter.xyz/reference/userreferrals) from the account that created the referral code: there you should see the newly referred account with zero points earned (this updates based on points distribution).
